# AI Career Recommendation System

## Project Overview
This represents a knowledge-based recommendation system designed to help computer science students find their best career path.
(Note: Source code is currently archived, showcasing system architecture and UI only).

## Features
- **Skill-based Matching:** Suggests roles based on user input (Python, Angular, etc.).
- **Interactive Chatbot:** Guides users through career choices.
- **Resource Suggestion:** Links to Coursera/Udemy courses.

## Tech Stack
- **Frontend:** Angular, Bootstrap.
- **Backend/AI:** Python, NLP Techniques.
